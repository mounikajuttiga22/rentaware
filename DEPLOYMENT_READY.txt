🐛 TROUBLESHOOTING

  Problem: Backend not starting
  Solution: Check Python 3.8+, install requirements.txt, check PORT 8000

  Problem: Frontend not loading
  Solution: Ensure backend is running, check browser console (F12)

  Problem: SMS not sending
  Solution: Verify Twilio credentials, check phone number verification

  Problem: Upload fails
  Solution: Use PDF/TXT format, check file size (<10MB)

  Problem: AI not responding
  Solution: Ollama is optional, system works with regex fallback
